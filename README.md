# WP.NetCore.VUE


#### WP.NetCore.VUE使用.NetCore+Vue前后端分离方式开发了一套搭配使用的后台管理系统
#### 实现了相关的基础功能，如：动态路由、策略授权、实时通信、任务调度、事件总线等相关基础功能
#### 并基于Nuxt.js框架开发了搭配使用的网站，通过SSR服务器端渲染生成静态页面。 

| 库  |  说明 |
| :------------ | :------------ |
|  StackExchange.Redis  | 缓存  |
|  Swagger  | 接口文档  |
|  Automapper  |  对象映射 |
|  AutoFac  | 依赖注入  |
|  AutoFac.DynamicProxy  | 动态代理  |
|  CORS   | 跨域  |
|  JWT   | 身份验证  |
|  Serilog  | 日志  |
|  SignalR   | 实时通讯  |
|  IpRateLimiting   |  限流 |
|  Quartz.net  |  任务调度 |
|  Cap  | 事件总线  |
|  RabbitMQ  | 消息队列  |
|  Polly  | 异常、超时重试  |
